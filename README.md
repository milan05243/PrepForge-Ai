## 📖 Overview

PrepForge AI is a comprehensive career preparation platform designed for students who want to strengthen their technical skills, improve interview performance, optimize resumes, and track their placement preparation journey.

The platform combines coding practice, DSA tracking, AI-powered resume analysis, mock interviews, and company-specific preparation resources into a single modern workspace.

---

## ✨ Features

### 📊 Smart Dashboard

* Real-time preparation analytics
* DSA progress tracking
* Quiz performance monitoring
* Daily streak system
* Resume analysis statistics
* Interview preparation insights

### 💻 Coding Arena

* Integrated Monaco Code Editor
* Coding problem practice
* Multiple language support

  * Python
  * Java
  * C++
* Run and submit solutions
* Competitive programming style interface

### 📚 DSA Tracker

* Topic-wise problem organization
* Progress visualization
* Personal notes for problems
* Problem completion tracking
* Direct Coding Arena integration

### 📝 AI Resume Analyzer

* ATS compatibility scoring
* Skill gap identification
* Resume improvement suggestions
* Resume evaluation history
* Role-based analysis

### 🎤 AI Interview Simulator

* Mock interview sessions
* Topic-based interview preparation
* Instant feedback and scoring
* Interview history tracking
* Technical and HR interview support

### 🏢 Company Preparation Hub

* Company-wise preparation roadmaps
* Eligibility criteria
* Recruitment process breakdowns
* Interview round details
* Placement preparation guidance

### 📘 Placement Preparation Hub

* DBMS preparation
* Operating Systems
* Computer Networks
* OOPs concepts
* Technical MCQs
* Interview-focused revision material
